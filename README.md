# A Game of Tic Tac Toe

This project is a Game of Thrones themed version of the classic tic toc toe game.

You can play the game by visit this URL --> https://tomdonohoe.github.io/tic-tac-toe/.

## How to play tic tac toe

 Tic tac toe is a game for two players, X and O. The players take turns marking a space in a 3Ã—3 grid. The first player to place three marks in a horizontal, vertical, or diagonal row is the winner. The game results in a draw if players both make the best moves.

In this version, X and O are replaced by one of your favourtie game of throne characters!
